# 🦋 Kelebek Sınav Editörü

Modern sınav ve öğrenci yönetim sistemi.

## Özellikler

📱 **Mobil Uygulama**
- WhatsApp-style navigation
- Swipe gesture (sağa kaydır = geri git)
- Öğrenci ve sınıf yönetimi
- Performans takibi

💻 **Web Arayüzü**
- Sınav editörü
- Çalışma kağıdı oluşturma
- Kelebek dağıtım
- PDF/Word/Excel export

☁️ **Google Drive Entegrasyonu**
- Otomatik yedekleme
- Cihazlar arası senkronizasyon

🔐 **Google Auth**
- Güvenli giriş
- Kullanıcı bazlı veri

## Kurulum

1. GitHub Pages'te otomatik deploy edilir
2. Google Cloud Console'da Authorized JavaScript origins'e ekle:
   ```
   https://KULLANICI_ADIN.github.io
   ```
3. Tarayıcıda aç ve Google ile giriş yap

## Teknik

- Tek HTML dosyası (self-contained)
- localStorage ile veri saklama
- Google Drive API
- Fabric.js canvas
- Responsive tasarım (mobil-first)
